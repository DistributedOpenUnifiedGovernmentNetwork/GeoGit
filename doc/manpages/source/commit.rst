geogit-commit documentation
#########################



SYNOPSIS
********
geogit commit -m <msg>


DESCRIPTION
***********

Stores the current contents of the index in a new commit along with a log message from the user describing the changes.

OPTIONS
*******

-m <msg>    Use the given <msg> as the commit message.
