geogit-push documentation
#########################



SYNOPSIS
********
geogit push [<options>] [<repository> [<refspec>...]]


DESCRIPTION
***********

Updates remote refs using local refs, while sending objects necessary to complete the given refs.

OPTIONS
*******

--all       Fetch all remotes.
